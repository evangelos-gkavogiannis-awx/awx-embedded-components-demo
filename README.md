# Airwallex Embedded Components Demo

Try Airwallex embedded components (KYC, KYC RFI, Beneficiary, Transfer, SCA) using your **demo** API credentials.

## Run

```bash
pnpm install
pnpm start
```

Open **http://localhost:3000**, enter your API Client ID and API Key, then launch a component.

## Requirements

- Node.js 18+
- [pnpm](https://pnpm.io)
- Demo API keys from [demo.airwallex.com](https://demo.airwallex.com)

Uses `api-demo.airwallex.com` (sandbox only).
